- [x] 支持设置 Github token 保存记录到云端

- [x] 支持按钮点击同步私有&公共资源
- [x] 支持快捷键唤起搜索框搜索片段，支持上下选择，回车选中增加到粘贴板

- [ ] 支持分享代码片段

## 数据结构定义

本地数据：

id: 唯一标识，uuid
name: 名字
desc: 描述信息
content: 内容
type: 类型
isPublic: 是否是公共
user: 用户信息
date: 添加时间
tags: 标签
isOwner: 是否是所有者，所有者可修改、删除等操作

可以用来搜索的内容：名字、描述、标签

线上公共库数据：

id: 唯一标识，uuid
name: 名字
desc: 描述信息
content: 内容
type: 类型
date: 添加时间
